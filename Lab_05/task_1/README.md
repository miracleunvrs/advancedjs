# Lab 5.1: Модернизация устаревшего кода


</div>

## Обзор

Модернизация устаревшего кода интернет-магазина с применением ES6+.

## Файлы

- `legacy_code.js` — исходный код с антипаттернами
- `legacy_analysis.md` — анализ проблем
- `modernization_documentation.md` — документация модернизации
- `modernized/` — модернизированные модули (config.mjs, utils.mjs, core.mjs, main.mjs)

## Что изменилось

| Было | Стало |
|------|-------|
| `var` | `const`/`let` |
| Функции-конструкторы | ES6 Classes |
| Публичные свойства | Private Fields `#` |
| Конкатенация `+` | Template Literals |
| `function() {}` | Arrow Functions |
| Глобальный код | ES6 Modules |

## Ключевые улучшения

**Инкапсуляция:** Приватные поля `#items` и `#price` защищают данные.

**Модульность:** Разделение на config, utils, core обеспечивает переиспользуемость.

**Читаемость:** Template literals и стрелочные функции делают код понятнее.
