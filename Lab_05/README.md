## Описание

Изучение современных возможностей JavaScript ES6+ и системы модулей. Модернизация устаревшего кода с применением современных стандартов и создание модульного приложения с различными стратегиями импорта.

## Задания

### Task 1 — Legacy Code Modernization

Модернизация устаревшего кода интернет-магазина с применением ES6+.

**Что было:**
- `var` для переменных
- Функции-конструкторы и прототипы
- Глобальное пространство имен
- Конкатенация строк через `+`

**Что стало:**
- `const`/`let` с блочной областью видимости
- ES6 классы с приватными полями `#`
- Модульная структура (config.mjs, utils.mjs, core.mjs, main.mjs)
- Template literals для строк

**Файлы:**
- `legacy_code.js` — исходный устаревший код
- `legacy_analysis.md` — анализ проблем
- `modernization_documentation.md` — документация модернизации
- `modernized/` — модернизированные модули

## Применяемые технологии

| ES6+ Feature | Использование |
|--------------|---------------|
| `const`/`let` | Вместо `var` |
| ES6 Classes | Вместо функций-конструкторов |
| Private Fields `#` | Инкапсуляция данных |
| Template Literals | Вместо конкатенации |
| Arrow Functions | Современный синтаксис |
| ES Modules | import/export |
| Dynamic Import | Ленивая загрузка |
