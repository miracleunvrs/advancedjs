# Архитектура приложения: Modular Calculator

## 1. Структура модулей
Приложение разделено на следующие части:

### Статические модули (загружаются сразу):
- `constants.mjs`: Текстовые сообщения и константы.
- `core.mjs`: Базовые арифметические операции.
- `utils.mjs`: Функции для работы с DOM.
- `app.mjs`: Главная логика, связывающая интерфейс.

### Динамические модули (загружаются по требованию):
- `scientific.mjs`: Сложные вычисления (синус, корень). Загружается при клике на "Advanced Mode".
- `history.mjs`: Модуль логирования истории. Загружается, когда блок истории становится видимым (IntersectionObserver).

## 2. Паттерны импорта
- **Named Imports**: Для функций из `core.mjs`.
- **Default Import**: Для основного класса калькулятора.
- **Module Object**: Для констант из `constants.mjs`.
