| Snippet | Pattern   | Level  |
|---------|-----------|--------|
| 01 | Singleton | Object |
| 02 | Factory   | Class  |
| 03 | Builder   | Object |
| 04 | Adapter   | Class  |
| 05 | Decorator | Object |
| 06 | Facade    | Object |
| 07 | Observer  | Object |
| 08 | Strategy  | Object |
| 09 | Command   | Object |
| 10 | State     | Object |

### Explanation Examples

- **Factory (Class level):** создание определяется наследованием.  
- **Adapter (Class level):** согласование интерфейсов через наследование.  
- **Decorator (Object level):** расширение поведения через композицию.  
- **Observer (Object level):** взаимодействие объектов по ссылкам.  
- **State (Object level):** переключение поведения через вложенные объекты-состояния.
